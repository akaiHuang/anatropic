Understanding this comparison

Why NFW always “fits better”

  • NFW uses 3 free parameters per galaxy (mass-to-light ratio + halo mass + concentration)
  • With 3 adjustable knobs, you can fit almost any smooth curve
  • This is not a prediction — it’s a retroactive fit. The halo parameters are chosen after seeing the data
  • Total free parameters for 175 galaxies: 525

Why Khronon is remarkable

  • Uses 1 free parameter per galaxy (mass-to-light ratio, physically constrained)
  • The acceleration scale a₀ = cH₀/(2π) is predicted, not fitted
  • One universal equation for all 175 galaxies — from dwarf irregulars to giant spirals
  • Getting χ²/dof ∼ 2–3 with 1 param where a 3-param fit gets ∼1 is a success, not a failure
  • Total free parameters: 175 (3× fewer than NFW)

How to read this chart

  • White dots: observed rotation velocities (SPARC)
  • Gray dashed: baryonic (visible matter) contribution alone
  • Blue line: Khronon prediction — a₀ fixed, only M/L fitted
  • Red line: best-fit NFW halo — 3 parameters fitted per galaxy
  • The blue line captures the shape of the rotation curve using only visible matter — that is the key test
